◆ご注意
    念のためウイルスチェックを行ってからご利用ください。

◆用途
    フォルダを監視し、変更があったファイルをバックアップします。
    バックアップの自動化や、作業履歴を残したいときにご利用ください。

◆動作確認
    Windows 10  64bit  22H2

◆インストール
    任意の場所にexeを置いて実行してください。
    同じ場所へ設定ファイルが作られます。

◆アンインストール
    exeファイルと設定ファイルを削除してください。

◆最小化
    最小化するとタスクトレイに格納されます。
    タスクトレイアイコンをクリックすると、通常サイズに戻ります。

◆バックアップ範囲
    『バックアップ元』　に指定したフォルダの直下のファイルのみです。
    サブフォルダはバックアップされません。
